                        🐛 TROUBLESHOOTING GUIDE
═════════════════════════════════════════════════════════════════════════

Issue: Server won't start
Solution: npm start (check port 5000 is available)

Issue: Login fails
Solution: Check credentials, create new account

Issue: CSS not loading
Solution: Hard refresh (Ctrl+F5), restart server

Issue: API returns 401
Solution: Login again, clear localStorage

Issue: Database connection error
Solution: Check MongoDB URI in .env file

Issue: Role-based redirect not working
Solution: Check user role in database, logout and login

See PROJECT_SETUP.md for detailed troubleshooting steps.
