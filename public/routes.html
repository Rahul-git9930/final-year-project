🚀 Quick Start Guide

1. Admin Testing

  1. Go to login page: http://localhost:5000
  2. Login with admin credentials (or signup with role "Admin")
  3. Redirected to: Admin Dashboard
  4. Access all management features

2. Student Testing

  1. Go to signup: http://localhost:5000/signup.html
  2. Create new account with role "Student"
  3. Login with new credentials
  4. Redirected to: Student Dashboard
  5. Access student features
