## 💻 Key Code Examples

### API Endpoints (7 total)
```javascript
POST   /api/payments/create-order              // Create order
POST   /api/payments/verify-payment            // Verify payment
GET    /api/payments/user-payments             // Get history
GET    /api/payments/:paymentId                // Get details
GET    /api/payments/admin/all-payments        // View all [Admin]
POST   /api/payments/admin/record-cash-payment // Record cash [Admin]
GET    /api/payments/admin/statistics          // Get stats [Admin]
```

### Frontend Methods
```javascript
paymentHandler.payFine(100, "Late Fine")
paymentHandler.payMembership(500, "Annual Fee")
paymentHandler.payPenalty(200, "Book Title")
paymentHandler.getUserPayments()
paymentHandler.getAllPayments()               // [Admin]
paymentHandler.recordCashPayment()            // [Admin]
paymentHandler.getPaymentStats()              // [Admin]
```

### HTML Integration
```html
<script src="/payment-handler.js"></script>

<!-- Student -->
<button onclick="paymentHandler.payFine(100)">💳 Pay Fine</button>

<!-- Admin -->
<button onclick="loadAllPayments()">📊 View Payments</button>
```
